This project is a platform game created in python using the pygame library. 
As a player you have to go to the next levels using teleporters. You also have to avoid obstacles placed on the levels.
To run this project you need to open it using Visual Studio Code (or another source code editor) and run main.py file (you need to download python on your computer).
You can also run this project using terminal, then you have to go to 'Platform_Game' folder and type 'python3/python main.py' (you need to download python on your computer).
